# 🤖 AI Chatbot using Python

A simple AI chatbot built using Python, NLTK, scikit-learn, and Streamlit.

This chatbot uses Natural Language Processing (NLP) techniques like tokenization, lemmatization, TF-IDF vectorization, and cosine similarity to understand user input and generate responses.

---

# 🚀 Features

- Simple chatbot interface
- NLP text preprocessing
- TF-IDF vectorization
- Cosine similarity matching
- Streamlit frontend
- Beginner-friendly project

# ▶️ How to Run

## Step 1: Install Libraries

pip install nltk scikit-learn streamlit

## Step 2: Download NLTK Data

python download_nltk.py

## Step 3: Run Chatbot

streamlit run chatbot.py
